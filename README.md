## 2차시
```python

a = []
a.append() #a리스트에 요소추가

a.extend([1,2,3]) #리스트를 이어 붙이기

a.remove() # 매개 변수로 입력한 데이터를 리스트에서 찾아 발견한 첫 변쨰 요소를 제거한다.

a.pop() # 리스트의 마지막 요소를 뽑아내어 리스트에서 제거

a.count(100) # 100이 a 리스트 안에 몇 개 존재하는지 센다.

a.sort() # 리스트를 정렬한다. 

a.reverse() # 리스트 내 요소의 순서를 반대로 뒤집습니다.



```
